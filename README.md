Simple template that "works on my machine":tm:

Versions used: 
- XCode 16.4
- CMake 4.2.1
- GLAD with OpenGL 4.1
- SDL 2.30.5
- GLM 1.0.3
- Apple clang version 17.0.0
- target: arm64
- c++20

For personal use, but feel free to copy if you are having a hard time setting up OpenGL and SDL on macos.
